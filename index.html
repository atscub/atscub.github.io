Things I built

El Alambique

News aggregator for Cuban independent media that avoids censorship by routing content through multiple popular services and clouds. It delivers through a mobile app with automatic endpoint discovery.

Boltdeck

A command center for Claude Code CLI. Not another chat GUI — the problem was managing dozens of sessions across projects. Browse, resume, read conversations, and diff files from one place.

Aurora Suite

End-to-end mail-to-web bridge that allowed Cubans to navigate the internet through email. Custom protocol, Android and Windows clients, ~4s round trips per page. 5k MAU, 100k emails/day at peak.

Nautapy

Python API and CLI for Cuba's Nauta captive portal with 41 stars and 20 forks on Github. Handles authentication and session management so you don't have to do it manually every time.

Food Fighters

2D browser fighting game with pixel art and chiptune audio. Built in ~19 hours using Claude Code with 5 specialized subagents as an experiment in autonomous multi-agent development.
